Estilos embebidos en las etiquetas HTML

Usar estilos CSS embedios en las etiquetas HTML es el peor y el menos utilizado de los metodos para agregar reglas de etilo. No es nada recomendable por la dificultad que implica realizar un cambio a posterior.

Esta forma de incluir CSS directamente en los elementos HTML solamente se utiliza en determinadas situaciones en las que se debe incluir un estilo muy especifico para un solo elemento concreto

Para insetar un estilo CSS en un elemento se utiliza la propiedad style de HTML, que en ningun momento se debe confundir con la etiqueta <style> que se introduce en la cebcera del documento
